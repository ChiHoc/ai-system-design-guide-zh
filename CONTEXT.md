# AI 系统设计指南中文镜像

本上下文定义中文镜像在翻译、同步和发布流程中使用的统一语言。

## Language

**上游原文**：
`ombharatiya/ai-system-design-guide` 的英文 `main` 分支，是内容事实与结构的唯一来源。
_Avoid_：原版、本体

**中文镜像**：
与上游原文路径和章节一一对应的简体中文译文集合，不包含本地化改编。
_Avoid_：中文版改编、中文分叉

**同步基线**：
已经被中文镜像完整表示并通过校验的上游 commit SHA。
_Avoid_：最新版本、同步时间

**翻译状态**：
记录每个上游 source blob 与对应 translation blob 的机器可读映射。
_Avoid_：翻译进度表

**同步批次**：
从一个同步基线到另一个上游提交之间，经翻译和校验后形成的单个 Pull Request。
_Avoid_：自动合并、定时发布
